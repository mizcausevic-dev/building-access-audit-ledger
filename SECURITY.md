# Security Notes

`building-access-audit-ledger` publishes synthetic proptech building-access and audit data only.

It does **not** contain:

- live resident or tenant data
- production badge identifiers
- proprietary controller configurations
- real vendor access windows
- real incident or audit packets

This repo is a public operator-surface proof for proptech and real-estate workflows. It is not a life-safety, regulatory, or security certification claim.
